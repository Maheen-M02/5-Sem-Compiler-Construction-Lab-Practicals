# 5-Sem-Compiler-Construction-Lab-Practicals

Here you can find all the lab practicals that I did while learning the compiler construction. It consists of the following codes:

Practical 1: Write a theory assignment for writing details about LEX and YACC compilation.

Practical 2: Count the number of comments, keywords, identifiers, words, lines, and spaces from an input file.

Practical 3: Count the number of words starting with the letter ‘A’.

Practical 4: Conversion of lowercase letters to uppercase and vice versa.

Practical 5: Conversion of decimal number to hexadecimal number in a file.

Practical 6: Test lines ending with the substring “com”.

Practical 7: Postfix Expression Evaluation.

Practical 8: Design a Desk Calculator with error recovery.

Practical 9: Write a parser for “FOR” loop statements.

Practical 10: Generate Intermediate Code (IC) for arithmetic expressions.
